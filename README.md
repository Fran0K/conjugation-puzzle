**Conjugaison Puzzle** 是一个互动的法语学习工具，旨在通过“拼图”的方式解构法语动词变位。通过将动词分解为逻辑块（词根 + 词尾），帮助学习者可视化并记忆 14 种法语时态的构建规律。


## ✨ 主要功能

*   **逻辑拆解**: 将变位分为 `词根 (Stem)` 和 `词尾 (Ending)`，复合时态包含 `助动词 (Auxiliary)`。
*   **拖拽交互**: 沉浸式的拖拽拼图体验，带有触觉反馈和视觉验证。
*   **全时态支持**: 覆盖 14 种时态，包括直陈式、虚拟式、条件式及其复合形式。
*   **多语言支持**: 界面及解释支持 🇫🇷 法语、🇬🇧 英语、🇨🇳 中文、🇯🇵 日语。
*   **智能提示**: 内置语法规则解释和基于当前动词的上下文提示。
*   **自定义练习**: 可以通过设置筛选特定的时态进行针对性训练。

## 🛠 技术栈

*   **前端**: React 18, TypeScript, Vite
*   **样式**: Tailwind CSS (自定义法语主题色), Lucide React (图标)
*   **后端/数据库**: Supabase (PostgreSQL + JSONB)
*   **数据管道**: Node.js + Google Gemini API (用于自动生成数据集)
*   **部署**: Cloudflare Pages
