# フランス語動詞活用ルール (日本語)

## 直説法現在形 (Présent)
- **構成**: 不定詞の語幹 + 現在形の語尾（3動詞群）
- **用法**: 現在行われている動作、習慣的な動作、または一般的な事実を表す。
- **例**: Je parle, je finis, je prends
- **詳細**:
  - *第1群動詞（-er）*: 不定詞から -er を取り除き、e, es, e, ons, ez, ent を付ける (例: je parle, nous parlons)
  - *第2群動詞（-ir）*: 不定詞から -ir を取り除き、is, is, it, issons, issez, issent を付ける (例: je finis, nous finissons)
  - *第3群動詞*: 動詞ごとに語幹や語尾が不規則に変化する (例: je prends, nous venons)

## 直説法半過去 (Imparfait)
- **構成**: 現在形 nous 形 − ons + 半過去の語尾
- **用法**: 過去の習慣的動作、背景描写、または過去のある時点で進行中だった動作を表す。
- **例**: Je parlais, je finissais, je prenais
- **詳細**:
  - *基本構成*: 現在形1人称複数から -ons を除き、ais, ais, ait, ions, iez, aient を付ける (例: nous parlons → je parlais)

## 直説法単純未来 (Futur Simple)
- **構成**: 不定詞 + 単純未来の語尾
- **用法**: 将来起こる動作や、未来に対する確実な判断を表す。
- **例**: Je parlerai, je finirai, je prendrai
- **詳細**:
  - *基本構成*: 不定詞または不規則な未来語幹 + ai, as, a, ons, ez, ont (例: parler → je parlerai)

## 直説法単純過去 (Passé Simple)
- **構成**: 語幹 + 単純過去の語尾
- **用法**: 文学的時制で、書き言葉において完結した過去の動作を表す。
- **例**: Je parlai, je finis, je pris
- **詳細**:
  - *用法*: 主に文学作品、物語文、正式な書き言葉で使用される (例: il entra, ils partirent)

## 条件法現在 (Conditionnel Présent)
- **構成**: 未来形の語幹 + 半過去の語尾
- **用法**: 仮定、条件の結果、推量、丁寧な依頼を表す。
- **例**: Je parlerais, je finirais, je prendrais
- **詳細**:
  - *基本構成*: 不定詞または未来語幹 + ais, ais, ait, ions, iez, aient (例: venir → je viendrais)

## 接続法現在 (Subjonctif Présent)
- **構成**: 現在形 ils 形 − ent + 接続法の語尾
- **用法**: 疑い、主観、感情、願望、必要性を表す。
- **例**: Que je parle, que je finisse, que je prenne
- **詳細**:
  - *基本構成*: 現在形3人称複数から -ent を除き、e, es, e, ions, iez, ent を付ける (例: ils parlent → que je parle)

## 接続法半過去 (Subjonctif Imparfait)
- **構成**: 単純過去の語幹 + 接続法半過去の語尾
- **用法**: 文学的時制で、過去における不確実または主観的な動作を表す。
- **例**: Que je parlasse, que je finisse, que je prisse
- **詳細**:
  - *用法*: 文学的・正式な書き言葉のみで使用され、現代口語では使われない (例: qu’il vînt, qu’ils eussent)

## 複合過去 (Passé Composé)
- **構成**: 助動詞 avoir / être 現在形 + 過去分詞
- **用法**: 完了した過去の動作で、現在と関連性のあるものを表す。
- **例**: J’ai parlé, je suis allé
- **詳細**:
  - *avoir を使用*: ほとんどの動詞で使用される。通常、過去分詞は主語と一致しない (例: j’ai fini)
  - *être を使用*: 移動・状態変化動詞およびすべての再帰動詞で使用され、過去分詞は主語と性・数一致する (例: je suis venu)

## 直説法大過去 (Plus-que-parfait)
- **構成**: 助動詞 avoir / être 半過去 + 過去分詞
- **用法**: 他の過去の動作よりも前に完了した動作を表す。
- **例**: J’avais parlé, j’étais parti
- **詳細**:
  - *avoir を使用*: 複合過去における avoir の用法と同じ規則で使用される (例: j’avais fini)
  - *être を使用*: 移動動詞・再帰動詞で使用され、過去分詞は主語と一致する (例: j’étais arrivé)

## 前未来 (Futur Antérieur)
- **構成**: 助動詞 avoir / être 単純未来 + 過去分詞
- **用法**: 未来のある時点より前に完了する動作を表す。
- **例**: J’aurai parlé, je serai parti
- **詳細**:
  - *avoir を使用*: 複合過去と同様に、大部分の動詞で使用される (例: j’aurai fini)
  - *être を使用*: 移動動詞・再帰動詞で使用され、過去分詞は主語と一致する (例: je serai venu)

## 直説法前過去 (Passé Antérieur)
- **構成**: 助動詞 avoir / être 単純過去 + 過去分詞
- **用法**: 文学的時制で、別の過去動作の直前に完了した動作を表す。
- **例**: J’eus parlé, je fus parti
- **詳細**:
  - *avoir を使用*: 大部分の動詞で使用され、文学的文脈に限定される (例: il eut fini)
  - *être を使用*: 移動動詞・再帰動詞で使用され、過去分詞は主語と一致する。文学語のみ (例: il fut venu)

## 条件法過去 (Conditionnel Passé)
- **構成**: 助動詞 avoir / être 条件法現在 + 過去分詞
- **用法**: 過去において実現しなかった仮定や条件の結果を表す。
- **例**: J’aurais parlé, je serais parti
- **詳細**:
  - *avoir を使用*: 条件が満たされていれば起こったはずの動作を表す (例: j’aurais fini)
  - *être を使用*: 移動動詞・再帰動詞で使用され、過去分詞は主語と一致する (例: je serais venu)

## 接続法過去 (Subjonctif Passé)
- **構成**: 助動詞 avoir / être 接続法現在 + 過去分詞
- **用法**: 疑い・感情・主観を伴う文脈で、すでに完了した動作を表す。
- **例**: Que j’aie parlé, que je sois parti
- **詳細**:
  - *avoir を使用*: 接続法の文脈で完了した動作を表す際に使用される (例: que j’aie fini)
  - *être を使用*: 移動動詞・再帰動詞で使用され、過去分詞は主語と一致する (例: que je sois venu)

## 接続法大過去 (Subjonctif Plus-que-parfait)
- **構成**: 助動詞 avoir / être 接続法半過去 + 過去分詞
- **用法**: 文学的時制で、過去に完了した仮定的・主観的な動作を表す。
- **例**: Que j’eusse parlé, que je fusse parti
- **詳細**:
  - *avoir を使用*: 接続法過去と同様の規則で使用され、文学語に限定される (例: qu’il eût fini)
  - *être を使用*: 移動動詞・再帰動詞で使用され、過去分詞は主語と一致する。文学語のみ (例: qu’il fût venu)
