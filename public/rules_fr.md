# Règles de conjugaison française

Ce document définit les règles de conjugaison française
**officielles et autoritaires** utilisées par le système **conjupuzzle**.

Seuls les temps explicitement documentés dans ce fichier
sont considérés comme **pris en charge par la logique du jeu conjupuzzle**.
Toutes les règles sont modélisées selon une approche compositionnelle
(Stem + Ending, ou Auxiliary + Past Participle).

---

## Présent de l’indicatif

- **Formule** : Stem + Terminaisons du présent
- **Usage** :  
  Exprime une action en cours, une habitude ou une vérité générale.

- **Exemples** :
  - parler → je parle  
  - finir → je finis  
  - prendre → je prends  

- **Détails** :
  - **Formation** :
    - 1er groupe (-er) : Stem + e, es, e, ons, ez, ent  
    - 2e groupe (-ir) : Stem + is, is, it, issons, issez, issent  
    - 3e groupe : stems et/ou terminaisons irréguliers

---

## Imparfait de l’indicatif

- **Formule** : Stem de l’imparfait + Terminaisons de l’imparfait
- **Usage** :  
  Décrit une habitude, une situation ou une action en cours dans le passé.

- **Exemples** :
  - parler → je parlais  
  - finir → je finissais  
  - prendre → je prenais  

- **Détails** :
  - **Formation** :
    - Stem = forme « nous » du présent sans *-ons*
    - Terminaisons : ais, ais, ait, ions, iez, aient

---

## Futur simple

- **Formule** : Stem du futur + Terminaisons du futur
- **Usage** :  
  Exprime une action future ou une certitude.

- **Exemples** :
  - parler → je parlerai  
  - finir → je finirai  
  - prendre → je prendrai  

- **Détails** :
  - **Formation** :
    - Stem = infinitif ou stem irrégulier
    - Terminaisons : ai, as, a, ons, ez, ont

---

## Conditionnel présent

- **Formule** : Stem du futur + Terminaisons de l’imparfait
- **Usage** :  
  Exprime une hypothèse, une conséquence conditionnelle
  ou une demande polie.

- **Exemples** :
  - parler → je parlerais  
  - finir → je finirais  
  - prendre → je prendrais  

---

## Subjonctif présent

- **Formule** : Stem du subjonctif + Terminaisons du subjonctif
- **Usage** :  
  Exprime le doute, l’émotion, le désir ou la nécessité.

- **Exemples** :
  - parler → que je parle  
  - finir → que je finisse  
  - prendre → que je prenne  

- **Détails** :
  - **Formation** :
    - Stem = forme « ils » du présent sans *-ent*
    - Terminaisons : e, es, e, ions, iez, ent

---

## Passé composé

- **Formule** : Auxiliary (avoir / être) + Participe passé
- **Usage** :  
  Exprime une action passée achevée ayant un lien avec le présent.

- **Exemples** :
  - parler → j’ai parlé  
  - aller → je suis allé  

- **Détails** :
  - **Avoir** : utilisé avec la majorité des verbes
  - **Être** : utilisé avec les verbes de mouvement et les verbes pronominaux
  - Accord du participe passé selon l’auxiliaire

---

## Plus-que-parfait

- **Formule** : Auxiliary (avoir / être) à l’imparfait + Participe passé
- **Usage** :  
  Exprime une action antérieure à une autre action passée.

- **Exemples** :
  - parler → j’avais parlé  
  - partir → j’étais parti  

---

## Futur antérieur

- **Formule** : Auxiliary (avoir / être) au futur simple + Participe passé
- **Usage** :  
  Exprime une action qui sera achevée avant un moment futur.

- **Exemples** :
  - parler → j’aurai parlé  
  - partir → je serai parti  

---

## Conditionnel passé

- **Formule** : Auxiliary (avoir / être) au conditionnel présent + Participe passé
- **Usage** :  
  Exprime une action passée hypothétique ou non réalisée.

- **Exemples** :
  - parler → j’aurais parlé  
  - partir → je serais parti  

---

## Subjonctif passé

- **Formule** : Auxiliary (avoir / être) au subjonctif présent + Participe passé
- **Usage** :  
  Exprime une action achevée liée au doute ou à la subjectivité.

- **Exemples** :
  - parler → que j’aie parlé  
  - partir → que je sois parti  

---
